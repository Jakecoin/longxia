# Tools

Longxia keeps the built-in tool surface, but removes runtime plugin expansion as a user workflow.

In this fork:

- use only the tools bundled into the deployment
- do not rely on runtime plugin installation
- ignore older examples that dock replies to Telegram, Discord, or WhatsApp

For deployment policy, see [Network Boundary](/design/network-boundary).
