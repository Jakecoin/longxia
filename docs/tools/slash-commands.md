# Slash Commands

Slash commands in Longxia are limited to the channels that remain enabled in this fork.

Legacy command behavior tied to Telegram drafts, Discord native commands, thread bindings, or channel docking to removed surfaces does not apply in Longxia.

Use the active channel guide for the supported channel you are configuring.
