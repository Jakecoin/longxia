# Hetzner

This page is reduced in Longxia.

Deploy on Hetzner only with the built-in Longxia channel set and without the upstream Telegram, Discord, or WhatsApp onboarding steps. Treat older examples for QR login, bot tokens, and external channel daemons as out of scope.
