# Fly.io

The upstream Fly.io guide assumed broader public messaging-channel access. Longxia does not follow that model.

Use Fly.io only for the reduced Longxia deployment surface and avoid reintroducing removed Telegram, Discord, or WhatsApp onboarding flows.
