# Chat Channels

Longxia keeps a reduced channel surface focused on controlled, reviewable integrations.

## Available in this fork

- [BlueBubbles](/channels/bluebubbles) — Recommended path for iMessage-style messaging.
- [Google Chat](/channels/googlechat) — HTTP webhook integration.
- [iMessage (legacy)](/channels/imessage) — Legacy macOS integration.
- [IRC](/channels/irc) — Classic IRC networks.
- [Signal](/channels/signal) — signal-cli based delivery.
- [Slack](/channels/slack) — Workspace app delivery.
- [WebChat](/web/webchat) — Built-in Gateway WebChat UI.

## Removed in Longxia

- Telegram
- Discord
- WhatsApp
- Plugin-delivered channel installs

Use only the channels shipped in this build or channels explicitly approved and bundled during deployment.
