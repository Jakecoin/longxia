# Channel Routing

Channel routing remains available in Longxia, but legacy examples that depended on Telegram, Discord, or WhatsApp identifiers were removed.

Use routing only with the channels still enabled in this fork, and derive channel-specific IDs from the currently supported provider docs and `openclaw channels capabilities`.
