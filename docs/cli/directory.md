# `openclaw directory`

Directory lookup remains available only for the channels still enabled in Longxia.

Legacy examples for Telegram, Discord, and WhatsApp were removed in this fork. Use the currently supported provider-specific identifier formats reported by:

```bash
openclaw channels capabilities
openclaw directory --help
```
