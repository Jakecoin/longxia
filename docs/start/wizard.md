# Wizard

The onboarding wizard in Longxia is narrower than upstream.

It focuses on:

- provider setup
- workspace/bootstrap files
- gateway configuration
- only the channels still enabled in this fork

Telegram, Discord, WhatsApp, and runtime plugin onboarding are not part of the Longxia setup path.
