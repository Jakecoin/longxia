# Setup

Use the setup flow in Longxia only with the built-in channels and preapproved capabilities included in this fork.

Older upstream steps that referenced WhatsApp QR login, Telegram bot tokens, Discord bot tokens, or runtime plugin installation are intentionally out of scope here.

For the current boundary, see [Network Boundary](/design/network-boundary).
