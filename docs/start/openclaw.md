# Personal Assistant Setup

This upstream guide was centered on WhatsApp, Telegram, and Discord. Those surfaces are not available in Longxia.

Use Longxia as a controlled-boundary deployment instead:

- choose only the channels still exposed in [Chat Channels](/channels/index)
- use preapproved bundled capabilities only
- do not rely on runtime plugin installation
- keep outbound integrations inside the reviewed deployment boundary

For the current policy, see [Network Boundary](/design/network-boundary).
