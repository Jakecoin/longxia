# Showcase

The upstream showcase page depended on public social distribution and third-party messaging surfaces. That content has been removed in Longxia.

Longxia examples should be documented through internal case studies or approved product documentation instead of public Discord, Telegram, or WhatsApp showcases.

For current product boundaries, see [Network Boundary](/design/network-boundary).
