# 聊天渠道

Longxia 只保留经过审查、边界明确的渠道能力。

## 当前可用

- [BlueBubbles](/channels/bluebubbles) — 推荐的 iMessage 类消息接入方式。
- [Google Chat](/channels/googlechat) — 基于 HTTP webhook 的接入。
- [iMessage（旧版）](/channels/imessage) — 旧版 macOS 集成。
- [IRC](/channels/irc) — 传统 IRC 网络。
- [Signal](/channels/signal) — 基于 signal-cli 的消息投递。
- [Slack](/channels/slack) — 工作区应用接入。
- [WebChat](/web/webchat) — 内置 Gateway WebChat 界面。

## 已在 Longxia 中移除

- Telegram
- Discord
- WhatsApp
- 通过插件安装的新渠道

这个分支只支持当前构建中已内置、且经过批准的渠道能力。
