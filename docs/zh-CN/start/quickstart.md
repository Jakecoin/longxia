# 快速开始

Longxia 不再提供以下快速开始路径：

- 通过 npm 全局安装后直接暴露外部渠道
- 通过二维码接入 WhatsApp
- 通过 Telegram 或 Discord 机器人令牌完成外联
- 通过运行时插件扩展新增渠道

## 建议流程

1. 在本仓库目录安装开发依赖。
2. 使用本地配置完成基础认证。
3. 启动 `dashboard` 或内置 WebChat。
4. 仅验证当前构建中已内置、已批准的工具与渠道。

## 当前原则

- 不引导到第三方聊天平台。
- 不引导到运行时插件安装。
- 不依赖外部安装包市场作为产品能力入口。
- 仅保留 Longxia 自己控制的服务边界。

如果你需要的是生产部署，请改看 Longxia 后续整理过的部署文档；上游的一键部署模板和第三方托管入口在本分支中不再作为推荐路径。
