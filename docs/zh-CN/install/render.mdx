# 在 Render 上部署

Longxia 不再提供基于 Render Blueprint 的默认部署方案。

上游页面会把 Render 作为快速上线和公网暴露入口，并将仓库、模板和第三方平台绑定在一起。对于 Longxia，这样的路径不再适合：

- 本分支不再围绕第三方外联渠道设计
- 本分支不再支持运行时插件安装
- 本分支不把第三方一键部署作为标准用户路径

## 当前建议

- 将 Render 视为你自己的基础设施选项，而不是 Longxia 官方推荐模板。
- 不直接使用上游仓库链接、按钮或模板参数。
- 只有在你能完全控制网络边界与配置审计时，才考虑托管部署。

如果需要托管版说明，应为 Longxia 单独编写并重新审查。
