# 在 Northflank 上部署

Longxia 不再提供 Northflank 模板部署文档。

上游的这类说明默认面向公网托管与外部渠道接入，而 Longxia 已经移除了：

- Telegram
- Discord
- WhatsApp
- 运行时插件扩展

因此，原始模板中的环境变量、公开 URL、以及后续渠道接入步骤都不应再作为本分支的推荐做法。

## 建议

- 使用你自己可审计、可控的部署流程。
- 将外部托管平台视为基础设施选择，而不是产品文档的一部分。
- 在没有单独审查之前，不要沿用上游的模板变量与公网入口说明。

如需正式部署，请等待 Longxia 专门整理的部署规范。
